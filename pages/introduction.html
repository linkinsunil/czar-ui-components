About

Czar-UI is HTML5 and CSS3 only library. Use it to build layouts and components. To know more about it, visit here.

CSS

Copy-paste the stylesheet <link> into your <head> before all other stylesheets to load Czar-UI's CSS and get started. 

                
            <link
              rel="stylesheet"
              href="../../css/main.css"
            />

JS

For proper functioning of some JS based components like Toast, copy-paste the following <script> above the closing </body> tag. 

                
            <script 
                src="../../js/main.js">
            </script>
